# ArtEcho: AI-Powered Interactive Art Appreciation

> **让名画活起来，让文化传下去**  
> 基于文心大模型的跨终端名画互动赏析平台

---

## 📌 项目简介

画语通_ArtEcho** 是一款融合AI、3D/4D情景再现与多模态交互的艺术科普工具，支持手机端与电脑端协同使用。用户可通过拍照或上传图片识别名画，获取拟人化语音讲解、全流程情景再现、深度艺术解析等沉浸式体验，致力于打造“人人可参与、处处可体验”的智慧艺术互动平台。

---

## ✨ 核心功能

### 🖼️ 名画识别与匹配
- **手机端**：拍照即识别，支持多角度、复杂光线环境
- **电脑端**：支持本地图片上传、网络链接、拖拽上传，支持批量识别
- **技术支撑**：文心大模型图像识别 + 专属名画数据库

### 🗣️ “名画开口”拟人化讲解
- 以名画为第一视角生成语音/文字讲解
- 支持多风格（亲切口语、专业学术、趣味故事等）与20+语言切换
- 电脑端支持讲解文本复制与导出

### 🎬 全流程3D/4D情景再现
- 3D动画还原创作过程，4D模拟流通与修复历程
- 手机端轻量化模型，手势交互流畅
- 电脑端高清渲染，支持全屏展示与视频导出

### 📚 名画深度信息库
- 基础信息、艺术解析、延伸知识、AI互动问答
- 电脑端支持导出PDF版“名画研究手册”
- 关联推荐同作者、同流派、同时期作品

### 🧠 个性化体验与多端同步
- 支持按年龄、兴趣定制内容
- 账号体系实现手机端与电脑端数据同步
- 教育版支持团队协作与共享文件夹

### 🏛️ 美术馆运营辅助模块
- 电脑端后台数据看板、内容管理、权限分配
- 手机端轻量化审核与数据预警
- 助力美术馆数字化运营与用户行为分析

---

## 🛠️ 技术架构

### 核心技术栈
- **基础模型**：文心大模型4.0（图像识别、NLG、TTS、问答）
- **开发平台**：文心快码、Vibe Coding
- **3D/4D引擎**：Unity + 轻量化模型压缩
- **数据来源**：权威艺术数据库 + 美术馆独家史料

### 系统流程
1. **数据层**：结构化名画知识库（多模态）
2. **识别层**：文心大模型图像识别接口
3. **生成层**：拟人化讲解 + 情景模型关联
4. **呈现层**：多端差异化展示 + 账号同步
5. **反馈层**：用户行为分析 + 运营报表

---

## 🚀 快速开始

### 手机端（APP）
1. 下载 ArtEcho APP
2. 注册登录或使用游客模式
3. 对准名画拍照或手动搜索
4. 选择讲解风格，开始沉浸体验

### 电脑端（Web）
1. 访问 [ArtEcho Web版]（开发中）
2. 注册登录
3. 上传图片（本地/链接/拖拽）
4. 查看高清情景与深度解析，支持导出与收藏
