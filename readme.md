# ArtEcho: AI-Powered Interactive Art Appreciation

> **让名画活起来，让文化传下去**  
> 基于文心大模型的跨终端名画互动赏析平台

---

## 📌 项目简介

**画展助手 ArtEcho** 是一款融合AI技术与多模态交互的艺术科普工具，支持多种方式获取图片并进行智能分析。用户可通过摄像头拍照、本地上传等方式获取图片，集成文心大模型进行图像识别和艺术分析，并将图片保存到个人画廊进行管理，致力于打造"人人可参与、处处可体验"的智慧艺术互动平台。

---

## ✨ 核心功能

### 📸 多方式图片获取
- **摄像头拍照**：实时拍摄，自动保存到画廊
- **本地图片上传**：支持拖拽上传、批量选择
- **移动端扫码**：二维码扫码，手机端便捷上传
- **智能识别**：基于AI技术的图像分析

### 🤖 AI图像识别分析
- **文心大模型集成**：调用百度文心大模型API
- **艺术品识别**：识别蒙娜丽莎、星月夜等名画
- **专业分析报告**：
  - 作品信息：标题、艺术家、创作时期
  - 技法分析：绘画技法、色彩运用、构图特点
  - 文化价值：历史意义、艺术地位、文化内涵
  - 历史背景：创作时间、收藏地点、历史故事
- **识别置信度**：显示AI分析的准确度
- **处理时间统计**：实时显示分析耗时

### 🖼️ 智能画廊管理
- **自动保存**：所有图片自动保存到个人画廊
- **搜索筛选**：按名称、描述、来源筛选图片
- **批量操作**：支持批量编辑、删除、导出
- **双视图模式**：网格视图和列表视图切换
- **拖拽上传**：画廊页面直接拖拽上传

### 🎨 图片编辑功能
- **重命名**：自定义图片名称
- **添加描述**：为图片添加详细描述
- **标签管理**：按来源自动分类标签
- **信息导出**：支持画廊数据导出

### 📊 统计分析
- **存储统计**：实时显示图片数量和存储占用
- **今日新增**：跟踪当日新增图片
- **来源分析**：按获取方式分类统计

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 网络连接（用于AI分析功能）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/yanxuefengyan/Hackathon_ArtEcho.git
   cd Hackathon_ArtEcho
   ```

2. **进入Web目录**
   ```bash
   cd web
   ```

3. **安装依赖**
   ```bash
   npm install
   ```

4. **启动开发服务器**
   ```bash
   npm run dev
   ```

5. **访问应用**
   打开浏览器访问：http://localhost:3000

### 构建部署

```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 🎯 使用指南

### 图片处理功能

1. **摄像头拍照**
   - 点击"📹 电脑摄像头"
   - 允许浏览器访问摄像头
   - 点击"📸 拍照"按钮
   - 图片自动保存到画廊

2. **本地图片上传**
   - 点击"📁 本地图片上传"
   - 选择单个或多个图片文件
   - 图片自动保存到画廊

3. **移动端上传**
   - 点击"📱 移动端上传"
   - 扫描二维码或复制链接
   - 在手机浏览器中访问

### AI图像分析

1. **单张分析**
   - 在预览区域点击"AI分析"按钮
   - 等待AI处理（约2-3秒）
   - 查看详细的艺术分析报告

2. **批量分析**
   - 在画廊中选中多张图片
   - 使用批量操作栏进行分析
   - 统一管理分析结果

### 画廊管理

1. **搜索和筛选**
   - 使用搜索框查找图片
   - 使用筛选按钮按来源分类
   - 切换网格/列表视图

2. **编辑管理**
   - 点击图片查看大图
   - 使用编辑按钮修改信息
   - 使用删除按钮移除图片

3. **高级功能**
   - 拖拽上传：直接拖拽图片到上传区域
   - 批量操作：选中多个图片进行批量编辑
   - 数据导出：下载包含所有图片信息的JSON文件
