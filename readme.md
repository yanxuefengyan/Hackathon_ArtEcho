# ArtEcho: AI-Powered Interactive Art Appreciation

> **让名画活起来，让文化传下去**  
> 基于文心大模型的跨终端名画互动赏析平台

---

## 📌 项目简介

**画展助手 ArtEcho** 是一款融合AI技术与多模态交互的艺术科普工具，支持多种方式获取图片并进行智能分析。用户可通过摄像头拍照、本地上传等方式识别名画，获取AI分析报告，并将图片保存到个人画廊进行管理，致力于打造"人人可参与、处处可体验"的智慧艺术互动平台。

---

## ✨ 核心功能

### 📸 多方式图片获取
- **摄像头拍照**：实时拍摄，自动保存到画廊
- **本地图片上传**：支持拖拽上传、批量选择
- **智能识别**：基于AI技术的图像分析

### 🖼️ 智能画廊管理
- **自动保存**：所有图片自动保存到个人画廊
- **搜索筛选**：按名称、描述、来源筛选图片
- **批量操作**：支持批量编辑、删除、导出
- **双视图模式**：网格视图和列表视图切换

### 🎨 图片编辑功能
- **重命名**：自定义图片名称
- **添加描述**：为图片添加详细描述
- **标签管理**：按来源自动分类标签
- **信息导出**：支持画廊数据导出

### 📊 统计分析
- **存储统计**：实时显示图片数量和存储占用
- **今日新增**：跟踪当日新增图片
- **来源分析**：按获取方式分类统计

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/yanxuefengyan/Hackathon_ArtEcho.git
   cd Hackathon_ArtEcho
   ```

2. **进入Web目录**
   ```bash
   cd web
   ```

3. **安装依赖**
   ```bash
   npm install
   ```

4. **启动开发服务器**
   ```bash
   npm run dev
   ```

5. **访问应用**
   打开浏览器访问：http://localhost:3000

### 构建部署

```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 🎯 使用指南

### 图片处理功能

1. **摄像头拍照**
   - 点击"📹 电脑摄像头"
   - 允许浏览器访问摄像头
   - 点击"📸 拍照"按钮
   - 图片自动保存到画廊

2. **本地图片上传**
   - 点击"📁 本地图片上传"
   - 选择单个或多个图片文件
   - 图片自动保存到画廊

3. **画廊管理**
   - 切换到"🖼️ 我的画廊"标签
   - 使用搜索框查找图片
   - 使用筛选按钮按来源分类
   - 点击图片查看大图
   - 使用编辑按钮修改信息

### 高级功能

1. **拖拽上传**
   - 在画廊页面直接拖拽图片到上传区域
   - 支持批量拖拽多个文件

2. **批量操作**
   - 点击图片选中多个图片
   - 使用批量操作栏进行批量编辑或删除

3. **数据导出**
   - 点击"📤 导出画廊"按钮
   - 下载包含所有图片信息的JSON文件

- **HTML结构**：单页面应用，模块化组件设计
- **CSS样式**：响应式设计，支持多设备适配
- **JavaScript功能**：
